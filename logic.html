Overview

Logic blocks automate tasks, process information and allow advanced controls. They include sensors that produce outputs when conditions are met, and logic gates that compute outputs based on their inputs. Most logic blocks are 1x1x2 and weigh 0.6 kg; the angle sensor and compass are larger and weigh 1.2 kg【542861138009269†L60-L71】.

Logic Gate Types

The game provides fifteen logic block types, including classic gates like AND, OR, XOR and NOR, plus sensors and math blocks【542861138009269†L85-L102】.

  • AND Gate: outputs only if all inputs are on【542861138009269†L118-L126】.
  • OR Gate: outputs if any input is on【542861138009269†L128-L135】.
  • XOR Gate: outputs if exactly one input is on【542861138009269†L137-L145】.
  • NOR Gate: outputs only if all inputs are off【542861138009269†L148-L155】.
  • Comparison Gate: compares sums of inputs for equality, greater-than or less-than【542861138009269†L195-L205】.
  • Accumulator: stores a numeric value that can accumulate over time.
  • Number Display: shows the numeric output of logic blocks.
  • Hue Light Panel: displays a coloured output based on inputs.

Sensors

Sensors detect conditions in the world and output signals:

  • Distance Sensor: outputs when an object comes within a specified distance【542861138009269†L161-L162】.
  • Speed Sensor: outputs when the speed of the block reaches a set value【542861138009269†L164-L167】.
  • Altitude Sensor: outputs when the block reaches a certain altitude【542861138009269†L169-L172】.
  • Gravity Sensor: outputs when gravity reaches a set amount【542861138009269†L174-L177】.
  • Angle Sensor: outputs when the block’s angle reaches a set amount【542861138009269†L179-L182】.
  • Compass: outputs when the facing direction falls within a specified angle relative to north【542861138009269†L184-L187】.

Using Logic Gates

Logic gates combine inputs to create outputs. For example, you can connect an AND gate to two sensors so that an engine activates only when both conditions are true. The gates clamp the sum of inputs between −1 and 1 and multiply by the gate’s output value【542861138009269†L105-L115】.

For complex mechanisms, combine logic gates with mechanical blocks and sensors. For example, create a proximity alarm by connecting a distance sensor to a NOR gate that triggers a light when no objects are nearby.
